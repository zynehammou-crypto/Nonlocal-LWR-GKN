# Graph Kernel Network for Nonlocal LWR Traffic Flow Model

## Description

This project aims to solve the nonlocal Lighthill-Whitham-Richards traffic flow model using Graph Kernel Networks.

## Objectives

- Study the nonlocal LWR model.
- Generate numerical solutions.
- Build graph representations.
- Train a Graph Kernel Network.
- Compare predictions with numerical simulations.

## Tools

- Python
- PyTorch
- PyTorch Geometric
- NumPy
- SciPy
- Matplotlib
